運用javascript設定亂數
使剪刀石頭布能夠隨機出拳
完成一個簡易的小遊戲
並設計滑鼠碰到圖片（按鈕）時圖片會變亮
提醒使用者現在選取的是哪張照片。

--

運用javascript來設定bmi計算機
只要使用者填入資訊就會自動計算目前的bmi值
並能將計算出的值於bmi值之間的格子反黑
並加入提醒字樣，讓計算機更有趣
若使用者漏填資料也會跳出提醒視窗提醒使用者將內容填完整
並且新增清除的按鈕，可以不用重新整理畫面就繼續計算。


--


運用Javascript設計一款色弱遊戲
遊戲設定在每個顏色只有十秒的時間能找不同顏色的色塊
越後面難度會越來越高，格子也會越來越多
並設定不讓玩家暫停遊戲；
遊戲結束後會統計獲得幾分並可選擇要直接再來一局還是回到首頁。
